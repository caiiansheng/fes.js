# Draggable 拖拽

# 概述

拖拽改变列表数据时使用, 处理数据顺序问题

# 代码示例

# API

# Draggable props

属性 说明 类型 默认值
disabled 是否阻止拖拽事件 Boolean false
list 数据列表，必填 Array

# Draggable Events

事件名 说明 回调参数
on-sort-ready 拖拽开始时发生，可以在这通过控制dragDisabled阻止事件 共三个参数：依次为鼠标事件对象、当前拖拽节点、当前拖拽节点下标
on-sort-end 拖拽结束时触发 共三个参数：依次为鼠标事件对象、当前拖拽节点、当前拖拽节点下标
